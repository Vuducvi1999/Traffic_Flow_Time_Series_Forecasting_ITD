Local File Execution Warning (CORS)

Because modern browsers block fetching file paths directly due to security restrictions (CORS), the dashboard could not load the data automatically. Vui lòng chạy lệnh sau trong thư mục dự án và mở đường dẫn tương ứng:

python -m http.server 8000

Sau đó truy cập: http://localhost:8000/dashboard.html
